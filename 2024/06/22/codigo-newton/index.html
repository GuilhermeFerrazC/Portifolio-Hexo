Código de Interpolação de Newton

A interpolação de Newton é uma técnica para encontrar o polinômio que passa por um conjunto de pontos dados. O código abaixo implementa este método em Python, permitindo calcular os coeficientes do polinômio interpolador e avaliar seu valor em um ponto específico.

Código

Explicação Breve

O código implementa a interpolação de Newton da seguinte maneira:

Função divided_diff:

  • Calcula os coeficientes das diferenças divididas, necessários para a construção do polinômio de Newton.

Função newton_interpolation:

  • Recebe três argumentos: x (lista de coordenadas x), y (lista de coordenadas y) e val (o valor de x para o qual queremos interpolar).
  • Usa os coeficientes das diferenças divididas para calcular o polinômio interpolador.
  • Retorna o valor interpolado e a expressão do polinômio.

Entrada de Dados:

  • O usuário é solicitado a inserir o número de pontos e os valores de x e y para cada ponto.
  • O valor de x para interpolação também é solicitado.

Saída:

  • O polinômio interpolador e o valor interpolado são exibidos.

  • ( y = [1, 4, 9] )
